# Student-Performance-Analyzer-Pandas
A Python project that analyzes student performance using Pandas. It reads student data from a CSV file, calculates grades, identifies toppers, performs subject-wise analysis, and generates a detailed performance report.

## Features

- Read student data from CSV
- Calculate total marks
- Calculate percentage
- Assign grades
- Find topper
- Find lowest scorer
- Subject-wise averages
- Highest scorer in each subject
- Lowest scorer in each subject
- Pass/Fail analysis
- Top 3 students
- Performance summary report

## Technologies Used

- Python
- Pandas

## How to Run

python main.py

## Dataset

students.csv
